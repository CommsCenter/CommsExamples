# Components

Custom components can be anything. They can be added to the page as normal action with pagebuilder.

Few examples:
 - counter
 - calendar
 - graph
 - game
 - slider

## Mixin `pckgComponent`
 - props:
  - `action` object
  - `content` object

## Build
Follow classic build procedure as descripted [here](./../README.md).
